# blog-webpage

You are going to create a blog website.  The main page will be called home.html and will have brief information about the website and you. You need to have a hyperlink to a second page and this will be the blog page. The blog will be a journal about your classes.  You will create a new article for each class last week.  You should use a section tag for each class you have (1st period section, 2nd period section, etc)

Create anchor links to the different course sections.
